비교 포인트
· A vs B — 같은 탭 구조에서 행 디자인 차이. A는 카드 경계·행 구분선·우측 정렬 컨트롤·행별 캡션, B는 현행 행 스타일 그대로.
· A vs C — 같은 카드 디자인에서 창 구조 차이. A는 탭당 1~2스크린 분량으로 끝나고, C는 카드화해도 세로 스크롤이 현행만큼 김 (왼쪽 현행 창과 높이 비교).
· 탭 구성(안): General = Refresh + Launch/Login + Version · Notifications = 알림 + 임계값 슬라이더 · Appearance = Menu Bar + Usage Jump + Weekly Chart.
